Heterogeneous Fenton

以固态铁基催化剂（Fe₃O₄、铁矿石等）替代均相铁离子，H₂O₂在催化剂表面活化产生·OH，避免铁泥问题，拓宽 pH 适用范围。

非均相芬顿反应机理

1
H₂O₂ → 催化剂表面吸附
H₂O₂ 经液相扩散到达催化剂表面，被表面活性位点捕获，形成表面络合物。
2
≡Fe²⁺ + H₂O₂ → ≡Fe³⁺ + ·OH + OH⁻
表面 Fe²⁺ 将 H₂O₂ 单电子还原，裂解 O-O 键，释放 ·OH（类均相芬顿反应）。
3
·OH + 有机物(R) → CO₂ + H₂O
·OH 以 10⁸~10¹⁰ M⁻¹s⁻¹ 速率攻击有机物，引发链式氧化降解直至矿化。
4
≡Fe³⁺ + H₂O₂ → ≡Fe²⁺ + ·OOH + H⁺
表面 Fe³⁺ 被 H₂O₂ 还原再生为 Fe²⁺，完成铁循环（速率限制步骤）。
5
≡Fe³⁺ + ·O₂⁻ → ≡Fe²⁺ + O₂
超氧自由基 ·O₂⁻ 也可参与还原 Fe³⁺，加速铁循环，提升 ·OH 产率。
🌀

非均相催化臭氧机理

1
O₃ → 催化剂表面吸附
气相/液相 O₃ 向催化剂表面扩散，被表面活性位点吸附，降低活化能。
2
≡M-OH + O₃ → ≡M-OOO· + H⁺
表面 Lewis 酸位点（≡M）或 Brønsted 羟基与 O₃ 反应，形成表面臭氧化物中间体。
3
≡M-OOO· → ≡M-O· + O₂
表面臭氧化物不稳定，分解产生表面活性氧 ≡M-O·，并释放 O₂。
4
≡M-O· + H₂O → ≡M-OH + ·OH
表面氧物种与水反应生成 ·OH，同时表面羟基再生（催化循环闭合）。
5
·OH + O₃ → ·O₂⁻ + H⁺ + O₂ （链式）
·OH 引发 O₃ 链式分解，产生更多 ·OH 和 ·O₂⁻，形成自加速氧化链。
